# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Give and receive feedback professionally.

## Unacceptable Behavior

- Harassment, trolling, or personal attacks.
- Discriminatory language or behavior.
- Sharing private information without consent.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

If you experience or witness unacceptable behavior, contact the maintainer through GitHub issues/discussions (or directly if contact info is provided).

## Scope

This code of conduct applies to project spaces and public interactions representing the project.
