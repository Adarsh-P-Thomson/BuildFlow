# Security Policy

## Supported Versions

Security fixes are applied to the latest published version.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead:

1. Open a private security advisory on GitHub for this repository, or
2. Contact the maintainer directly if a private contact channel is available.

Include:

- description of the issue
- impact assessment
- reproduction steps or proof of concept
- suggested mitigation if known

We will acknowledge reports as soon as possible and work on a fix.
